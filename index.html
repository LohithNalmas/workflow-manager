Connect Supabase

One-time setup to link your database

Quick Setup (2 minutes):

1

Go to supabase.com → Create a free account → New Project

2

In your project: go to Project Settings → API and copy your Project URL and anon/public key

3

Go to SQL Editor and run the setup SQL below, then paste your credentials here

📋 Click to view required SQL (run in Supabase SQL Editor) 
-- Run this in Supabase SQL Editor:

CREATE TABLE IF NOT EXISTS records (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  type TEXT NOT NULL,
  invoice_number TEXT,
  party_name TEXT,
  item_count INTEGER,
  status TEXT DEFAULT 'pending',
  picker_id TEXT DEFAULT '',
  picker_name TEXT DEFAULT '',
  checker_id TEXT DEFAULT '',
  checker_name TEXT DEFAULT '',
  picking_done_at TIMESTAMPTZ,
  checking_done_at TIMESTAMPTZ,
  packing_done_at TIMESTAMPTZ,
  delivered_at TIMESTAMPTZ,
  employee_name TEXT,
  employee_role TEXT,
  employee_level TEXT,
  employee_available BOOLEAN DEFAULT true,
  employee_present BOOLEAN DEFAULT true,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- Enable real-time for the table:
ALTER TABLE records REPLICA IDENTITY FULL;

-- Allow public access (adjust for production):
CREATE POLICY "allow_all" ON records FOR ALL USING (true) WITH CHECK (true);
ALTER TABLE records ENABLE ROW LEVEL SECURITY;

Credentials are saved in your browser's localStorage only.
